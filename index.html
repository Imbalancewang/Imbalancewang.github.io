Basic skills of plot

In this artical, I will use the data “mtcars” in the R language itself to draw many kinds of pictures. So let’s have a preliminary understand of R plot.

You can see the concreate code here.
I will give you a brief introduction of the function plot.

plot

You can use function plot to get the first and the last figures above, which you just need to modify parameters to get.

  • Usage: plot(x,y,…)
  • Arguements: x,y are the datas you need to input.
    “…” means the other parameters you need to modify.
    For some examples, “main=title” means an overall title for the plot. “sub” means a subtitle for the plot. xlab means a title for x axis. You can see more parameters with the word help(“plot”) in R.

So let’s see a concreate example.

attach(mtcars)

opar<-par(no.readonly=TRUE)

par(lty=2,pch=2)

plot(head(wt),head(mpg),type="b",main="wt-mpg")

par(opar)

detach(mtcars)

I put the codes in the pages below the figure, and I will explain the codes lines and lines. The first and last lines mean attach the data “mtcars” and detach the data. The second line is to generate a list of current graphical parameters that can be modified, which means you must use it. And the third line, of course, is to modify the parameters. Lty is the abbreviation of the type of line. Lty=2 means the dotted line. Pch=17 is the point character, which means the solid triangle. The following pictures show the usage of parameters.


Of course, the parameters is more than just pch and lty, such as lwd, cex and so on. The lwd means the width of line and the cex means the size of symbol. You can refer to the help docement for more specific information.

what’s more, the R also can modify the axis by yourself. You can clearly see the figure above and the code is here. By the code, we can know the line axis(4,at=x,labels = x,col.axis="blue",las=2,cex.axis=0.7,) that controls the output of axis. The parameter in the code is shown below.

  • 4 stands for the position. 1 is the bottle. 2 is the left side. 3 is the up side. 4 is the right side.
  • at means the scale of the axis.
  • las=2 means the word is perpendicular to the axis and las=0 means the word is parallel to the axis.
    If you want to know more about the parameter of the axis, you can refer to the document of help.

Basic Information

R as a statistical analysis software, is a statistical analysis and graphical display. It runs on the operating systems of UNIX, Windows, and Macintosh, and embeds a very convenient and practical help system, as compared to other statistical analysis software. R also has following characteristics:

  • R is free software, which also means it is completely free, open source and you can download any setup program and source code in the web.
  • R provides graphical facilities that are printed directly on a computer or on paper for data analysis and display.
  • R language has efficient data processing and storage facilities.
  • R provides a large, consistent and integrated set of tools for data analysis.

In conclusion, R is most widely used programming language in the world. It is first choice for data scientists and is supported by a dynamic and talented contributor community. The R language is taught at university and deployed in key business applications.
