Solving PDE by matlab(I)

Posted on 2017-06-07 | In June 2017

Brief Introduction

As we all know, the partial differential equation(PDE) is very hard to get the analytic solution in some way. But we can get numerical solution by matlab. Thus I will solve the problems such as elliptic PDE ,hyperbolic PDE and so on. I will give the concrete examples as follows.

Concrete Steps

In matlab, there are two simple methods to solve the PDE. One is using the pdetool which is suitable for solving two-dimensional equation, and the other is to use pdepe function. So let’s focus on the first method at the beginning. The pdetool is very easy to operate, for it is a GUI which has a friendly interface and simple operation. You just need to obey the following advice.

  • Make sure what kind of PDE you want to solve like elliptic PDE or what problems you are faced with including heat conduction equation and others.
  • Determine the boundary condition.
  • Clear the geometry of the PDE.
  • Partition finite element and solve the equation.

Example for Elliptic PDE

Problem:

  • First of all, just input “pdetool” in command window in matlab to open the pde toolbox.

  • And you should draw a unit circle and click the boundary mode or click the omega Icon. The final result is shown in the figure

  • Select the command of Boundary -> Specify boundray cindition, and choose the Dirichlet condition with the h=1,r=0 set.

  • Click the PDE button and then a dialog box which is shown below will pop out.

  • Partition finite element.click the triangle button and the double-triangle button.

  • Finally click the “=” button to solve the equation. And the numerical solution for Elliptic PDE will presented in your eyes.

Gradient Descent Method(I)

Basic Information

The gradient descent method is an optimization algorithm, also known as steepest descent. The steepest descent method is one of the simplest and oldest methods for solving unconstrained optimization problems.

Solving Process

The calculation of the gradient descent method is to solve the minimum value in the direction of the gradient descent (and also to solve the maximum value along the ascending direction of the gradient).
$a_(k+1)=a_k+\rho_k*s_. s_k$represents the gradient negative direction, while the \rho_k stands for the search step in the gradient direction.Gradient direction, we can get the derivative of the function, the step size is more difficult to determine, too big, it may be divergent, too small, convergence is too slow.The general method of determining the step size is determined by the linear search algorithm, which treats the coordinates of the next point as a function of a_k, and then satisfies the minimum of F(a_k+1).

Kmeans Algorithm

Basic Information

K-means is a typical clustering algorithm based on distance, using the distance as a similarity evaluation index. The closer between two objects, the greater similarity is. It is considered that the clusters are made up of objects which are close to each other, so the compact and independent clusters are chosen as the final target.

The selection of K initial cluster centers has a great influence on the clustering results, because the first step of the algorithm is to select any k objects randomly as the center of the initial clustering, and represent a cluster. In each iteration of the algorithm, each object remaining in the data set is re assigned to the nearest cluster based on the distance from each cluster center. When all data objects are examined, an iterative operation is completed, and a new cluster center is calculated. If the value of J does not change before and after an iteration, the algorithm is proved to be convergent.

Algorithm Process

Input: The numbers of clusters k, and the database containing N data objects.

Ouput: K clustering with minimum variance criterion.

  • (1) K objects are selected as initial cluster centers from N data objects.
  • (2) According to the mean (central object) of each cluster, calculate the distances between each object and the central object. And also, on the basis of minimum distance, the corresponding objects are divided.
  • (3) Recalculate the value of each (with change) clustering (center object).
  • (4) Cycle (2) to (3) until each cluster does not change.

Algorithm Advantages

  • The algorithm is simple.
  • It is efficient and scalable for large data sets.

Algiruthm Disadvantages

  • In the K-means algorithm K is given in advance, the selection of this K value is very difficult to estimate.
  • In the K-means algorithm, it is necessary to determine an initial partition according to the initial cluster center, and then optimize the initial partition. The selection of the initial cluster center has a great impact on the clustering results. Once the initial value is not good, it may not be able to get effective clustering results, which has become a major problem of the K-means algorithm.
  • From the framework of K-means algorithm can be seen, the algorithm needs to constantly adjust constantly calculating the sample classification, a new clustering center adjusted, so when there is a very large amount of data, the algorithm time overhead is very large.
